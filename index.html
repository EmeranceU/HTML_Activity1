Introduction

Python is a high-level, interpreted programming language known for its clear syntax and readability. It was created by Guido van Rossum and first released in 1991.

Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming. It has a large standard library that covers many areas such as web development, data analysis, artificial intelligence, and scientific computing.

Some key features of Python include:

  • Easy to learn and use with a clean, readable syntax
  • Interpreted language, no need to compile before running
  • Dynamically typed, you don't need to declare variable types


Getting Started

To start using Python, you need to install it on your computer. You can download the latest version from the official Python website at python.org.

Once installed, you can open a terminal or command prompt and type the following command to verify the installation:

python --version

You can run Python code interactively in the Python shell by typing:

python

Or you can write your code in a file with a .py extension and run it from the terminal.


Variables and Data Types

In Python, variables are created the moment you assign a value to them. There is no need to declare the type of a variable beforehand.

name = "Alice"
age = 25
height = 5.6
is_student = True

Python has several built-in data types, including:

  • str — text strings like "Hello"
  • int — whole numbers like 10
  • float — decimal numbers like 3.14
  • bool — True or False values
  • list — ordered collections like [1, 2, 3]

You can check the type of any variable using the built-in type() function.


Control Flow

Python uses indentation to define blocks of code. The most common control flow statements are if, elif, and else.

age = 18

if age >= 18:
    print("You are an adult.")
elif age >= 13:
    print("You are a teenager.")
else:
    print("You are a child.")

Unlike many other languages, Python does not use curly braces to define code blocks. Instead, consistent indentation (usually 4 spaces) is required.

Python also supports comparison operators such as ==, !=, >, <, >=, and <= for making decisions in your code.


Functions

Functions in Python are defined using the def keyword. They allow you to group reusable pieces of code together.

def greet(name):
    return "Hello, " + name + "!"

print(greet("Alice"))

Functions can accept parameters and return values. If no return statement is provided, the function returns None by default.

Python also supports default parameter values, which allow you to call a function without providing all arguments:

def greet(name="World"):
    print("Hello, " + name + "!")
     greet()         # prints: Hello, World!
     greet("Alice")  # prints: Hello, Alice!

Lists and Loops

Lists are one of the most versatile data types in Python. They can hold items of different types and are defined using square brackets.

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Output: apple

You can loop through a list using a for loop:

for fruit in fruits:
    print(fruit)

Python also has a while loop, which runs as long as a condition is true:

count = 0
while count < 5:
    print(count)
    count += 1

Lists come with many useful built-in methods such as:

  • append() — adds an item to the end of the list
  • remove() — removes the first matching item
  • sort() — sorts the list in ascending order
